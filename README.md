# NnsDao-Frontend-template-React

## Project setup

```
nvm install && nvm use
npm ci
yarn add @psychedelic/plug-connect

```

### Compiles and hot-reloads for development

```
dfx start
dfx deploy
chmod -R 755 .dfx
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
